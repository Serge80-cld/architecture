\# Lab 01 – Setup du Lab Modern Workplace



\## Objectifs

\- Préparer l’environnement pour les labs suivants.

\- Vérifier les prérequis Intune / Entra / MECM.



\## Prérequis

\- Tenant Microsoft 365

\- Entra ID P1/P2

\- Intune activé

\- Un device Windows 10/11 pour tests



\## Étapes

1\. Vérifier l’accès au portail Intune

2\. Configurer RBAC minimal

3\. Créer les groupes dynamiques

4\. Préparer un device de test

5\. Vérifier les endpoints réseau Microsoft



\## Validation

\- Accès OK

\- Groupes dynamiques OK

\- Device visible dans Entra ID
